ENC Converter 아이콘 시안

가장 추천하는 방향은 A입니다. 문서 + 코드 + 변환 화살표 조합이라 앱 용도가 가장 직관적으로 보입니다.

Concept A

문서 중심형. 코드 파일을 UTF-8로 정리하는 앱이라는 메시지가 가장 직접적입니다.

Concept B

변환 과정 강조형. 결과 문서 쪽에 UTF-8 배지를 직접 넣어 인코딩 변환 앱임을 더 분명하게 보이게 했습니다.

Concept C

기술 유틸리티형. 변환 엔진이나 인코딩 코어 같은 인상을 주는 좀 더 도구적인 방향입니다.
